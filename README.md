# WhatsappMessageSender
Automatic message sending script

This is a script to send the whatsapp messages through the web interface.

Libraries used:

1. Selenium

It is used to interact with the web browser for testing and automation purposes.

The script takes the following as input:

1. The name of the chat

2. The message to be sent 

The user is required to scan the whatsapp web QR code on their phone for the script to work as expected.

The script can be used to prank with friends by spamming the chat with messages.
